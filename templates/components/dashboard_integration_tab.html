Customize the Widget

The embed automatically uses your saved configuration—no code editing required. Just make sure the following tabs are updated before you paste the snippet:

  • Appearance tab: Colors, avatar, launcher shape, greeting text.
  • Prompt tab: Persona, tone, response instructions.
  • Knowledge Base: Uploaded files, crawled pages, FAQ entries.

Once you click “Save” on each tab, those settings flow directly into the widget—simply paste the snippet into your site and the latest branding + behavior will follow.

Test the Integration

After installing the snippet, verify the widget in a real environment:

  1. Open the page on desktop and mobile, confirm the launcher appears, opens, and closes.
  2. Send at least one test question and confirm it uses your latest knowledge base + prompt.
  3. Ensure the button position and colors match your brand on all breakpoints.

help Frequently Asked Questions

Do I need a website ID?
No! Since you have 1 chatbot per account, the script automatically uses your chatbot configuration. Just paste the code snippet and it will work.
Can I customize the widget's colors to match my brand?
Yes, advanced customization is available. Contact support for help with custom CSS integration.
Does the widget work on mobile devices?
Yes, the widget is fully responsive and works on all modern mobile browsers.
How do I track widget performance?
Conversation metrics and widget analytics are available in the Overview and System Health sections.
What if I need to remove the widget?
Simply remove the code snippet from your website. The widget will no longer load on subsequent visits.
