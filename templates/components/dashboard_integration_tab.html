Customize the Widget

The embed automatically uses your saved configuration—no code editing required. Just make sure the following tabs are updated before you paste the snippet:

  • Appearance tab: Colors, avatar, launcher shape, greeting text.
  • Prompt tab: Persona, tone, response instructions.
  • Knowledge Base: Uploaded files, crawled pages, FAQ entries.

Once you click “Save” on each tab, those settings flow directly into the widget—simply paste the snippet into your site and the latest branding + behavior will follow.

Test the Integration

After installing the snippet, verify the widget in a real environment:

  1. Open the page on desktop and mobile, confirm the launcher appears, opens, and closes.
  2. Send at least one test question and confirm it uses your latest knowledge base + prompt.
  3. Ensure the button position and colors match your brand on all breakpoints.

help_center Frequently Asked Questions

help_outline

Do I need a website ID?

No! Since you have 1 chatbot per account, the script automatically uses your chatbot configuration. Just paste the code snippet and it will work.
help_outline

Can I customize the widget's colors to match my brand?

Yes! Use the Appearance tab to customize colors, avatar, and suggested messages. All changes are automatically reflected in the widget. For advanced CSS customization, contact support.
help_outline

Does the widget work on mobile devices?

Yes, the widget is fully responsive and works on all modern mobile browsers (iOS Safari, Chrome, Firefox, etc.). The chat interface adapts automatically to screen size.
help_outline

How do I track widget performance?

Conversation metrics and widget analytics are available in the Overview and System Management sections. Monitor usage, response times, and user interactions.
help_outline

What if I need to remove the widget?

Simply remove the code snippet from your website. The widget will no longer load on subsequent visits. Your chatbot configuration remains saved in the dashboard.
help_outline

Which LLM providers are free to use?

Free Tier Available: Google Gemini and Groq offer free tiers with generous limits. You can use these providers without any cost for testing and small-scale deployments. Check the LLM Configuration tab to configure these providers.
help_outline

Which LLM providers require payment?

Paid Providers: OpenAI (GPT models), Anthropic (Claude), DeepSeek, Together AI, and other premium providers require API keys with billing enabled. Pricing varies by provider and model. Configure your API keys in the LLM Configuration tab.
help_outline

How do I switch between LLM providers?

Go to the LLM Configuration tab, select your preferred provider from the dropdown, enter your API key (if required), and click "Save Configuration". You can test the connection before saving to ensure it works correctly.
help_outline

How does the Knowledge Base work?

The Knowledge Base uses RAG (Retrieval Augmented Generation). Upload documents, crawl websites, or add FAQs. After ingesting, the chatbot uses this information to answer questions accurately. Enable AI Content Cleaning for better results by removing navigation and boilerplate content.
help_outline

What file types are supported for upload?

Supported formats: PDF, TXT, DOC, DOCX, CSV. Maximum file size is 16MB. Files are processed and can be previewed before ingesting to the knowledge base. Use categories to organize your documents.
help_outline

How do I update my chatbot's knowledge?

Upload new documents, crawl updated website pages, or add/edit FAQs in the Knowledge Base tab. After ingesting, the chatbot immediately uses the new information. You can also delete outdated content to keep your knowledge base current.
help_outline

Can I customize the chatbot's personality?

Yes! Use the Prompt Editor tab to customize the chatbot's personality, tone, and response style. Choose from preset templates or create your own custom prompt. The chatbot name can be set in the Appearance tab.
help_outline

Is my data secure and private?

Yes! Each user has an isolated knowledge base and configuration. Your uploaded files, crawled URLs, FAQs, and API keys are stored separately and are not accessible to other users. All data is encrypted and secured.
